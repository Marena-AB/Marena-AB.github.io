Mirrulations Search

Client capstone

Federal regulations search platform · Built with clients from CareSet and Merck

An open-source search platform that helps researchers and policy professionals navigate federal rulemaking across dockets, documents, and public comments from regulations.gov. Working in a real agile environment with sprint releases and code review, I designed and built the full document download system from scratch: REST endpoints, a Redis-backed async worker packaging files to S3, PostgreSQL schema changes, and EC2 deployment with systemd. Also shipped Google OAuth, CFR part filtering, and multi-field sorting.
